OpenShift Origin M4

Grab the latest OpenShift Enterprise M4 installer here:

sh <(curl -s https://install.openshift.com/origin-m4)

Popular Options

Want to get fancy? You can pass options to the installer by adding them to the end of the command, like this:

sh <(curl -s https://install.openshift.com/) -e -s rhsm -u user@company.com

Here are some useful settings:

            -a, --advanced-mode              Enable access to message server and db server customization.
            -c, --config-file FILEPATH       The path to an alternate config file
            -w, --workflow WORKFLOW_ID       The installer workflow for unattended deployment.
            -e, --enterprise-mode            Show OpenShift Enterprise options (ignored in unattended mode)
            -s, --subscription-type TYPE     The software source for installation packages.
            -u, --username USERNAME          Red Hat Login username
            -p, --password PASSWORD          Red Hat Login password
            -d, --debug                      Enable debugging messages

You can also download the installer and run it from a CD or USB drive.

Origin

Origin is the upstream community project for OpenShift Enterprise. There's no better place to develop your new feature and have it battle tested. The best way to try out the latest code is with the all-in-one Docker image.
